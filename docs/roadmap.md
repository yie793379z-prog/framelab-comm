# Roadmap

This roadmap keeps FrameLab scoped as a lightweight, open-source communication research workflow tool.

## v0.1

Current MVP:

- paste-based text import
- multi-sample workspace
- four built-in communication studies templates
- manual coding form
- mock suggestions by default, with optional real AI mode
- CSV, JSON, and Markdown export
- English and Simplified Chinese interface

## v0.2

Usability and documentation:

- stronger onboarding for first-time users
- clearer import guidance and example-driven UI help
- small accessibility improvements
- more polished export formatting
- better project docs and contributor guidance

## v0.3

Richer schema editing and project workflow:

- improve project save/load flow beyond the current JSON export/import
- expand project-level codebook editing beyond display text only
- add clearer named-project workflow and project metadata controls
- support safer field-level schema customization without breaking saved coding rows
- improve dataset import for CSV and structured text

## v0.4

Expanded AI configuration and provider options:

- refine the current optional real AI workflow
- support more configurable provider and model options
- improve privacy and cost messaging in AI-enabled setups
- keep strong human-in-the-loop controls
- add stronger validation and explainability around AI suggestions

## Good First Issue Ideas

- improve bilingual wording for one template
- add one more fictional example dataset
- polish Markdown export formatting
- add a small onboarding hint for blank-line sample separation
- improve mobile layout for the coding form
