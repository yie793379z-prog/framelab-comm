# Research Disclaimer

FrameLab is a learning and research-assistance tool for communication and media studies workflows.

## What FrameLab Is

- a lightweight workspace for trying coding schemas
- a tool for organizing and exporting early research material
- a student-friendly environment for small projects and coursework

## What FrameLab Is Not

- a final authority on coding decisions
- a replacement for methodology training
- a substitute for careful reading, theoretical grounding, or instructor feedback

## AI Assistance

AI suggestions in FrameLab are editable aids, not final academic judgments.

Users remain responsible for:

- reviewing every suggested code
- correcting weak or misleading suggestions
- explaining final coding choices in assignments, memos, or theses

By default, FrameLab uses local mock suggestions for this feature.

If a user enables optional real AI mode with their own API key:

- selected sample text is sent to the configured AI provider
- `.env.local` should stay local and must not be committed
- API usage may incur costs on the user's own account
- providers may occasionally return temporary high-demand or unavailable errors
- FrameLab may retry once and then fall back to mock suggestions for continuity
- the output should still be treated as tentative and editable

## Responsibility for Research Quality

FrameLab can help structure work, but research quality still depends on the user.

That includes:

- choosing appropriate samples
- applying a defensible coding scheme
- documenting decisions clearly
- checking reliability and consistency where relevant

## Data Caution

The current MVP is local-first, but optional real AI mode uses an external API provider.

Users should avoid uploading sensitive, private, or identifiable data into AI-enabled workflows unless they fully understand the privacy, ethics, institutional, and cost implications involved.
